# Simple Cave Adventure

A simplified text-based adventure decision-making game written in C for beginners, where your choices determine your fate in a small cave system.Easy to play.

## Overview

"Simple Cave Adventure" is a basic decision-making adventure game designed for beginners. Navigate through a simple cave, make critical decisions, and discover whether you'll find treasure or meet your doom.

This game is built entirely with basic conditional statements (if/else) and simple loops, demonstrating fundamental programming concepts for beginners.

## Features

- Pure decision-based gameplay using simple if/else statements
- Basic branching storylines based on your choices
- Health and treasure tracking systems
- Simple combat system with strategic choices
- Straightforward game mechanics
- Beginner-friendly code structure

## How to Play

1. Clone or download the repository
2. Compile the game: `gcc -o game game.c`
3. Run the executable: `./game`
4. Follow the prompts and enter your choices when prompted
5. Experience how your decisions shape your adventure!

## Game Mechanics

The game operates on simple decision-making:
- At each stage, you're presented with numbered choices
- Your selection determines the next sequence of events
- Different paths lead to different outcomes
- Your health and treasure are affected by your choices

## Code Concepts Demonstrated

This simple game demonstrates basic programming concepts:
- Variable declarations and assignments
- Basic input/output operations
- Conditional statements (if/else)
- Loops (while)
- Simple arithmetic operations
